1. prisma init  -> npx prisma init
2. change DB url in .env file
3. Create models in ./prisma/schema.prisma
4. Run migration -> npx prisma migrate
5. install prisma client -> npm i @prisma/client
6. create db.mjs file and write config for prisma client.
7. generate prisma client files -> npx prisma generate
8. import db.mjs file in your controller and use.
9. add all commands for future use in ./package.json
